# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/bbaecd97-f1d3-4e23-87db-5344d09218be

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

## Firebase setup

This project is now wired to Firebase Hosting + Firestore for homepage data.

1. Install dependencies:
   `npm install`
2. Create `.env.local` and set values from `.env.example`:
   - `VITE_FIREBASE_API_KEY`
   - `VITE_FIREBASE_AUTH_DOMAIN`
   - `VITE_FIREBASE_PROJECT_ID`
   - `VITE_FIREBASE_STORAGE_BUCKET`
   - `VITE_FIREBASE_MESSAGING_SENDER_ID`
   - `VITE_FIREBASE_APP_ID`
3. Set your Firebase project id in `.firebaserc` (`default` value).
4. Login and deploy:
   - `npm run build`
   - `npm run firebase:deploy`
